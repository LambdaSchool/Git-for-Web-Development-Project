    1. What is Semantic HTML? 
    Semantic HTML are tags like header, article, or aside used to reinforce the structure of a webpage.
    2. What is HTML used for? 
    HTML is used to create the structure or bones of a webpage.
    3. What is an attribute and where do we put it? 
    An attribute is an additional properties like src or href, it is put at the end of a tag like <img src=””> or <a href=””></a>
    4. What is the h1 tag used for? How many times should I use it on a page?
    h1 tag are header tags and should be used once on a webpage.
    5. Name two tags that have required attributes
    Img tag and a tag

    A class is an attribute used to specify a class name in css style sheet.
    11. What elements can I add a class to? 
    Any element can have a class attribute.
    12. How many times can I use the same class on a page? 
    As many times you need
    13. How do I get my link to open in a new tab?
    Use the target attribute and set it to “_blank”

    It’s used as a text placeholder if the image doesn’t show up.

    A Section tag is semantic while a div tag is non-semantic.
    17. What is CSS used for? 
    CSS is used to style and decorate a webpage.

An id is used to identify one single element for example if every div has a class but one div needs to be styled differently then an id can be used for that div like <div class= “classname” id= “idname”> </div>
    
    20. How do we select classes in CSS?
    To select a class in CSS use a period first then write the name of the class like .classname{}
    21. How do we select a p element with a single class of “human””?
    .human p{}
    22. What is a parent child selector? When would this be useful?
    Selects every element that is the child of a specific parent element. It is useful when a parent element has many child elements.
    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
    Pseudo selectors are used to specify when the css should activate like hover or active.
