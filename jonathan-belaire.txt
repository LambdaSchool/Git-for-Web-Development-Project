1. What is Semantic HTML?
Semantic HTML elements have clear meanings as to how it would be used as well as its function.

2. What is HTML used for?
HTML is used to give a web page a structure where content can be displayed and specialized code can be ran to facilitate the user experience.

3. What is an attribute and where do we put it?
An attribute is used in HTML as a way to modify or add specificity to an element. To use an attribute, it must be inside of the <>, following the name of the element.

4. What is the h1 tag used for? How many times should I use it on a page?
The <h1> tag should be used only once to indicate the most important headline of a webpage.

5. Name two tags that have required attributes
<input> & <select>
 
6. What do we put in the head of our HTML document?
Metadata, title, external stylesheets and scripts.

7. What is an id?
An id is an attribute that assigns a unique identifier to an HTML element that can later be referenced.

8. What elements can I add an id to?
All HTML elements.

9. How many times can I use the same id on a page?
Only once.

10. What is a class?
A class is an attribute that can be used to reference a class in a stylesheet.

11. What elements can I add a class to?
All HTML elements.

12. How many times can I use the same class on a page?
There is no limit.

13. How do I get my link to open in a new tab?
<a href="#" target="_blank">Link</a>

14. What is the alt attribute in the image tag used for?
In the event that the image cannot be displayed, the browser will display the value of the alt attribute.

15. How do I reference an id?
For a style declaration, the use of a "#" before the name will reference the element with its id. This can also be done in JavaScript with the use of document.getElementById('id');

16. What is the difference between a section and a div
The <section> element is considered to be a semantic element while a <div> is not.

17. What is CSS used for? 
CSS is used to add visual styling to a webpage.

19. What is the difference between a class and an id? - Give me an example of when I might use each one
A class is typically used to style a webpage in CSS, while an id is typically used to manipulate a webpage using JavaScript.

22. What is a parent child selector? When would this be useful?
A parent child selector is used to assign css properties to all of the elements that are child of a parent element. This is useful when a developer wishes to style only specific elements of their webpage using this method.

23. How do you select all links within a div with the class of sidebar?

24. What is a pseudo selector?
A pseudo selector is used to define a specific state of an element, which can have its own styling; mouseover, click, visited, focused, etc.

Version control allows you to revert back to a series of save points when and if the need arises. It is also an extremely efficient way to share your work with others.

35. How often should I commit to github?
As often as necessary; fixed a bug, corrected a typo, or created a feature that works among other things.

37. Walk me through Lambda's git flow.
Step 1: Fork to personal Github account.
Step 2: Copy the URL from the repository.
Step 3: Open Terminal
Step 4: Clone repository with "git clone repo-url"
Step 5: Change directory with "cd repo-name"
Step 6: Create branch from the original with "git branch firstName-lastName"
Step 6: Change branch with "git checkout firstName-lastName"
Step 7: Add all files in current directory to staging area with "git add ."
Step 8: Commit all changes and attach a message with "git commit -m 'message'"
Step 9: Push all files to GitHub with "git push origin firstName-lastName"
Step 10: Submit a pull request to submit work.

Stretch Questions

1. What is the difference between an inline element and a block element?
Block-level elements start a new line and (by default) occupy as much space as it possibly can to the left and right. Inline elements do not start a new line and only occupy as much space as necessary.   

2. What happens when an element is positioned absolutely?
The element is positioned absolutely, relative to its first positioned parent.

3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
element {display: inline-block;}

4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
Block: <div> <p> <ul>
Inline: <span> <a>
Inline-Block: <img>

5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them?
The box model is used for layout and consists of margins, borders, padding, as well as the content. The "fix" for the box model is to use "element {box-sizing: border-box;}".
