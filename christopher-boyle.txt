    1. What is Semantic HTML? 
	-Semantic HTML tags are tags that define what some section of text is.

    2. What is HTML used for? 
	-HTML is used to mark up text for a web browser to interpret and 
	display.

    3. What is an attribute and where do we put it? 
	-Attributes are given to elements to modify them. we up them within
	the <> after the element name (ie. <a href="#"> with href being the 
	attribute.).

    4. What is the h1 tag used for? How many times should I use it on a page?
	-h1 is used for the most important title of the page and should only be
	used once per page.

    5. Name two tags that have required attributes
	-<a> and <img> both require attributes.

    6. What do we put in the head of our HTML document? 
	-We put h1 tags, an intro paragraph, and navigation information in the 
	head of the HTML document.

    7. What is an id?
	-An id is an attribute that gives an element a unique id.

    8. What elements can I add an id to? 
	-An id can be added to any element.  

    9. How many times can I use the same id on a page? 
	-An id can only be used once.

    10. What is a class? 
	-A class is an attribute that can be added to elements to grab them to
	be styled in CSS.

    11. What elements can I add a class to? 
	-Any type of element can have a class added to it.
	
    12. How many times can I use the same class on a page? 
	-As many times as you'd like.

    13. How do I get my link to open in a new tab?
	-To open a link in a new tab use the target attribute.

    14. What is the alt attribute in the image tag used for? 
	-The alt attribute shows alternative text in the case that the viewer
	of the web page cannot load the image.

    15. How do I reference an id?
	-An id is referenced by using a # followed by the id.

    16. What is the difference between a section and a div
	-A section is used to wrap different parts of a webpage together for a 
	specific purpose while div and can used to wrap anything.

    17. What is CSS used for? 
	-CSS is used to style a webpage. For example the font type, font size, 
	background color, or margin size
