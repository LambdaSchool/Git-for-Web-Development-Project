Review Questions:
    1. What is Semantic HTML? 
    Semantic code is naming HTML tags in a way that is clear to the coder and the user. It uses element names that are descriptive and intuitive so there is no guess work. Examples would be <form> <table> <section> <header>.

    2. What is HTML used for? 
    HTML stands for Hyper Text Mark-up Language and is used to build the structure of any webpage and its content.

    3. What is an attribute and where do we put it? 
    Attributes add another layer of descriptive information to HTML tags. We put it right next to the element in the tag. For example <img></img> is a basic image tag, <img src="aPicOfMe.jpg"></img> adds a path to a jpg photo in your HTML page.

    4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is used to identify the title of page and set order to the page. H1 is given priority in the HTML hierarchy and can only be used once on a page.

    5. Name two tags that have required attributes.
    2 tags that have required attributes are:
    <img src="example.jpg"></a>
    <a href = "example.com"/>
    all image tags need an image attribute to display the image, all hyperlink tags need a hyperlink attribute to link to.

    6. What do we put in the head of our HTML document? 
    Meta tags & links to other source sheets.

    7. What is an id? 
    An id is an HTML attribute used to style and set hierarchy to the page. Unlike a class, an id can only be declared once. 

    8. What elements can I add an id to? 
    An id can be used on any HTML element.

    9. How many times can I use the same id on a page? 
    Only once.

    10. What is a class? 
    A class is an HTML attribute which can be assigned to any element in order to use in CSS to add style on top of order.

    11. What elements can I add a class to? 
    Any HTML element.

    12. How many times can I use the same class on a page? 
    As many as needed.

    13. How do I get my link to open in a new tab?
    Add a target="_blank" attribute in the tag element and it will open up in a new tab.

    14. What is the alt attribute in the image tag used for? 
    It is used to name what the image is in case the image cannot be seen or displayed.

    15. How do I reference an id?
    In HTML it is referenced as such:
    <h1 id="myHeader">My Header</h1>
    In CSS it is called out by a #:
    #myHeader {

    }

    16. What is the difference between a section and a div.
    A section is used for semantics, div is used for style. 

    17. What is CSS used for? 
    CSS or Cascading Style Sheets are used to add style to an HTML page.

    18. How to we select an element? Example - every h2 on the page
    h2 {}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one.
    Class is a way to style any css element. A class can be used more than once - or as many times as needed. an ID is also used to style, but can only be used once on a page.
