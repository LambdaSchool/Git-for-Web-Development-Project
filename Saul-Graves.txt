1. What is the difference between an inline element and a block element?
    A block element will encapsulate all of the content of a certain div, and stylization will apply to all of the content within the block element. 
    An inline element can be applied to a specific piece of content within a block element, and can be stylized differently from the rest of the block element. 
    An example is making a single sentence bold/italicized within a larger paragraph that is not. 
2. What happens when an element is positioned absolutely? 
    The element is positioned to the closest ancestor element. The ancestor element needs to be positioned, otherwise the absolutely positioned element will use the document body.
