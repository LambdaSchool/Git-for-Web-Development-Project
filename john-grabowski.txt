 1. What is Semantic HTML? 
Semantic HTML are HTML tags that explain their purpose on the page. They are mostly used to help the developer stay organized, and to help SEO. Examples are (<header><footer><section>)

 2. What is HTML used for? 
HyperText Markup Language is used to provide the foundation for web pages. This is typically the text and line breaks that other languages like CSS and Javascript build on top off.

3. What is an attribute and where do we put it? 
Attributes provide additional information about the elements, and they are placed within the opening tag of an element.

4. What is the h1 tag used for? How many times should I use it on a page?
h1 is a header tag and it is used to distinguish the first section of text on a webpage (usually). It should only be used once since using multiple can mess with SEO.

5. Name two tags that have required attributes
<img> tag and the <a> tag (src and href respectively)

6. What do we put in the head of our HTML document?
The charset we are using and the document title

 7. What is an id? 
An id is a unique tag you can put on a single element. CSS and JavaScript can then access the element with that id tag and change it.

8. What elements can I add an id to? 
Any element.

9. How many times can I use the same id on a page? 
once

10. What is a class? 
serves similar function to id but can be put on multiple different elements at once. Mainly used by CSS where id is mainly used by JavaScript

11. What elements can I add a class to? 
any element

12. How many times can I use the same class on a page? 
As many times as I please

13. How do I get my link to open in a new tab?

14. What is the alt attribute in the image tag used for? 
If for some reason the picture can not be displayed it will instead show the alt text, which should include a short description of the picture.

 19. What is the difference between a class and an id? - Give me an example of when I might use each one
id's should only be used when referencing one element, and typically you might use one when you need to affect an element with JavaScript. Classes on the other hand can be put on however many elements or tags you want. These are typically used when styling with CSS

22. What is a parent child selector? When would this be useful? 
A parent child selector matches an element that is a that is a direct child of another. This would be useful, for instance, if you wanted to grab a <p> tag that was the child of a <div>, and not other <p> tags.

23. How do you select all links within a div with the class of sidebar?

24. What is a pseudo selector?
a pseudo selector defines a particular state of another element, and adjusts that element when the state is met.

git checkout -b 'firstname-lastname'
make your changes to the files
git add .
to add the changes from this directory to your branch and prepare them for committing
git commit -m 'commit message'
comits or "saves" the changes made to your branch to the clone of the repo.
git push -u origin branch-name
pushes all your commits from local machine to github.
go to your forked repo of the project on github
submit pull request
