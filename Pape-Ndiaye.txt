1. What is Semantic HTML?

Semantic HTML is the use of HTML markup to reinforce the meaning of the information on the page.

    2. What is HTML used for?

HTML is a markup language used to structure a webpage.

    3. What is an attribute and where do we put it? 

An attribute is a special word used to control an elements behaviour and is placed in the opening tag.

    4. What is the h1 tag used for? How many times should I use it on a page?

The h1 tag is the main heading tag and should only be used once.

    5. Name two tags that have required attributes

<img> and <a>

    6. What do we put in the head of our HTML document? 

The <head> element which contains the metadata of the page.

    7. What is an id? 

An id is an attribute placed in the opening tag of an element used for specificity.

    8. What elements can I add an id to? 

Any element.

    9. How many times can I use the same id on a page? 

Once.

    10. What is a class?

Similar to an id, a class is an attribute placed in the opening tag for specificity, only difference is that it can be used more than once.

    11. What elements can I add a class to? 

Any.

    12. How many times can I use the same class on a page? 

As many times as you need.

    13. How do I get my link to open in a new tab?

Add the target="blank" attribute after the link address.

    14. What is the alt attribute in the image tag used for?

An alternate text for an image if the image cannot be displayed.

    15. How do I reference an id?

With the # sign.

    16. What is the difference between a section and a div?

A scetion is used to partition a page like headers, footers, etc. the div tag is used to divide a document where no other tag makes sense.

    17. What is CSS used for?

CSS is used to style a page.

    18. How to we select an element? Example - every h2 on the page.

Simply type the elements tag name.

    19. What is the difference between a class and an id? - Give me an example of when I might use each one

A class can be used multiple times whereas an id can only be used once. You might want to use an id for something like a button or a a paragragh. A class would be better if you wanted to reference all the headings on a page.

    20. How do we select classes in CSS?

With the . sign.

    21. How do we select a p element with a single class of “human””?

.human {}
    22. What is a parent child selector? When would this be useful? 

This references an element nested inside another element. It's useful when you want to specifcally reference an element within an element.

    23. How do you select all links within a div with the class of sidebar?

}
    24. What is a pseudo selector?

A keyword added to a selector that gives it a special state.
