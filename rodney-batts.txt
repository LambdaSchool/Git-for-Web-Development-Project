1. What is Semantic HTML? 
tags  1. What is Semantic HTML? 
HTML tags that by their label have meaning and relevance to the content theirin. They are read by and are relevant to both human and machine readers.
    
    2. What is HTML used for?
HTML is used to create the structure of web pages. Its tags allow the items within the tags to be exhibited as directed.

	14. What is the alt attribute in the image tag used for?
       It identifies what the image would be when the image isn't rendered on the page.
	   
    15. How do I reference an id?

    16. What is the difference between a section and a div
    A section is semantic. It can be read by a search engine. It has meaning by virtue of its name; it means that content therin is grouped whereas a div per se has not defined meaning.
	
	17. What is CSS used for?
CSS is used to give style to a page already structured by HTML.
	
    18. How to we select an element? Example - every h2 on the page
	By enclosing descriptors within braces following the element in the CSS script of stylesheet.
	For example, we would select every h2 on a page by writing h2 in front of the braces that contain the attributes desired for that descriptor.
    
    19. What is the difference between a class and an id? - Give me an example of when I might use each one 
        A class identifies an element that we might want to have a certain characteristic wherever it appears. For example, all h2 elements are red, all h3 elements are blue.
	An  id identifies one specific element. For example, we want the 2nd section to have a pink background color.
	A class might be more complex than one element though. The class might be the 1st and 3rd div in each section.
	
	20. How do we select classes in CSS?
	A dot . preceeds the class name. Example: .Myclass.
	
    21. How do we select a p element with a single class of “human””?
	.human.
	p.human {}
    22. What is a parent child selector? When would this be useful?
This is a greater than angle bracket > between the parent tag and the immediately nested child tag.
This is great for selecting an element immediately nested within the parent tag.
	
    23. How do you select all links within a div with the class of sidebar?
	div .sidebar a {}
	
    24. What is a pseudo selector?
	Usually a class that selects elements that are in a specific state ie a:hover wherein some property is invoked upon hover.
