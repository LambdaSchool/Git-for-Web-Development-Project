    1. What is Semantic HTML?
Semantic HTML is to give order to web pages/code. It is stuff like <section>, <main>, <header>, <aside>, <footer>, etc.
It tells browsers what each portion of a web page is for and what it is.
    2. What is HTML used for?
Hyper Text Markup Language. Used to mark up, organize, and structure the content of web pages.
    3. What is an attribute and where do we put it?
The attribute typically goes inside the opening tag of an element and gives more information about the element (e.g. id=name).
    4. What is the h1 tag used for? How many times should I use it on a page?
h1 tag is used to create large headings (h1 is most important/largest, h6 is least important/smallest). Use one time per page.
    5. Name two tags that have required attributes
<form><input>
    6. What do we put in the head of our HTML document?
Meta data such as title, name, authors, description, keywords, viewport information, charset=utf-8, and emoticon for tabs.
    7. What is an id?
A unique # identifier for a particular element; it is placed in the opening tag.
    8. What elements can I add an id to?
Unique things that you don't want to be paired with anything else. <div>,<span>,<p>,<h#>. You can add one to anything.
    9. How many times can I use the same id on a page?
One time, period.
    10. What is a class?
A unique identifier for one or more elements.
    11. What elements can I add a class to?
<div>,<span>,<p>,<h#>. You can add one to anything.
    12. How many times can I use the same class on a page?
Multiple times. No limit.
    13. How do I get my link to open in a new tab?
<a href="#" target="_blank">
    14. What is the alt attribute in the image tag used for?
alt is used to display meta data about the image in the case that the browser cannot render the image or for accessibility
purposes for those that are blind.
    15. How do I reference an id?
Use a # in CSS, then the id name (e.g. #name).
    16. What is the difference between a section and a div?
A section partitions the web page into smaller portions for the web browser to understand the content more fully. A <div>
is used mainly for styling in CSS; <div>s are not displayed.
    17. What is CSS used for?
To style a web page and make it responsive and beautiful.
    18. How do we select an element? Example - every h2 on the page?
h2 {....}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one?
Class is for selecting multiple elements, wherease an id is for one unique element. Class could be used for certain things
you want styled the same, like layers in a form or something, and an id is for something specific, like one portrait image
that is unlike all other images on the page.
    20. How do we select classes in CSS?
.class {....}
    21. How do we select a p element with a single class of “human””?
p .human {....}
    22. What is a parent child selector? When would this be useful?
 Parent-child selector is like the p .human as mentioned above. It is usefule to select a very specific thing amid an already
established grouping.
    23. How do you select all links within a div with the class of sidebar?
.sidebar a {....}
    24. What is a pseudo selector?
It is basically used to identify a unique state of an element, such as when one hovers over text, or clicks inside a box.

This is important because it allows me to see the history of changes that I have made to a program or website. It helps me to
go back to a previous state of the app/site before an error occurred, assuming that I put sufficient notes so I know what was
going on in the previous version(s).
